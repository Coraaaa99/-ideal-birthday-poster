📖 使用说明

👨‍💼 管理员（您）：

  1. 在"设置"页面输入管理员密码（默认：admin123）解锁管理员模式
  2. 进入"上传海报模板"页面，上传每月的海报模板
  3. 配置祝福语的位置、颜色和字体大小
  4. 返回"设置"页面，在云端同步区域输入 JSONBin API Key 并保存
  5. 点击"上传配置到云端"（首次上传会生成固定 Bin ID）
  6. 将 Bin ID 复制并告知销售人员（或者直接内嵌到网页中）

👥 销售人员（普通用户）：

  1. 打开网页后，系统会自动从云端同步最新模板和贴纸
  2. 进入"生成祝福海报"页面，输入车主姓名，选择月份和模板
  3. 可选：调整文字样式和贴纸
  4. 点击"生成海报"，然后下载

🔐 安全提示：

建议修改默认管理员密码，防止其他人修改您的配置。
